# Boitas - Authentic Odia Delicacies

A modern, elegant website showcasing Boitas' authentic Odia delicacies with a focus on heritage and craftsmanship.

## About

Boitas celebrates Odisha's culinary roots with a modern twist. This website features:

- **Chenna Pie** - Odisha's classic dessert, redesigned and handcrafted
- **Khaja Bites** - Heritage sweet of Odisha in bite-sized form
- Beautiful illustrations showcasing Odia culture and heritage
- Responsive design with elegant typography and smooth animations

## Tech Stack

- **React** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **Lucide React** for icons

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/shrixtacy/Boitas-Web.git
   cd Boitas-Web
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and visit `http://localhost:5173`

## Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.
